# 🚀 DealDrop — Smart Product Price Tracker

DealDrop is a modern SaaS-style web application that allows users to track product prices from any e-commerce website and get notified when prices drop. Built with performance and simplicity in mind, it helps users save money effortlessly.

---

## 🌐 Live Demo

👉 https://deal-drop-smoky.vercel.app

---

## ✨ Features

- 🔍 Track prices from any product URL  
- ⚡ Fast and reliable scraping using Firecrawl API  
- 📉 Detect real-time price changes  
- 🔔 Smart alert-ready system for price drops  
- 🧠 AI-powered structured data extraction  
- 💻 Clean, responsive, and modern UI  
- 📊 Dashboard to manage tracked products  

---

## 🛠 Tech Stack

- **Frontend:** Next.js, Tailwind CSS  
- **Backend:** Next.js API Routes  
- **Scraping Engine:** Firecrawl API  
- **Database:** PostgreSQL (NeonDB)  
- **Deployment:** Vercel  

---

## 📌 How It Works

1. User enters a product URL  
2. Backend scrapes the webpage using Firecrawl  
3. Extracts structured data:
   - Product Name  
   - Current Price  
   - Currency  
   - Product Image  
4. Stores the product data in database  
5. Displays tracked products in a dashboard
